# 🎸 Band Name Generator

A simple Python program that generates fun and creative band names based on user input.

## 💡 Features
- Takes user input (city name and pet name)
- Combines inputs to generate a unique band name
- Simple and interactive command-line program

## 🛠️ Tech Used
- Python

## ▶️ How to Run
1. Install Python
2. Run the file:
   python BandNameGenerator.py

## 📌 Example
Hello User
Enter name of city you grew up in
Mumbai
Enter name of your pet
Tommy
Your band name is: MumbaiTommy
